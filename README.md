# Realtor Landing Page (Vanilla)

Small, responsive landing page scaffold for a realtor focused on lead capture (free valuation).

Files:
- `index.html` — main markup and JSON-LD
- `styles.css` — mobile-first styles
- `script.js` — form handling and analytics hooks
- `images/` — add hero/agent/sold images here

Quick start:
1. Open `index.html` in your browser (no server required) or use a static server for nicer dev workflow.

Optional local server (Node.js):
```
npx http-server . -c-1
```

Next steps:
- Replace placeholders ([City], phone, images) with real content.
- Wire the form to your backend or Netlify Forms.
- Add analytics (GA4 / GTM) and consent banner if required.
